# Registro de decisiones

## DEC-0001 — Desarrollo como dispositivo médico

**Fecha:** 2026-07-21  
**Estado:** Aprobada  
**Decisión:** HydroSeat será desarrollado como dispositivo médico terapéutico.  
**Impacto:** Se aplicará regulatory-by-design, ISO 13485 como QMS principal e ISO 14971 para riesgos.
